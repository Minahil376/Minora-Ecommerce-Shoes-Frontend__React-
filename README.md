# 👟 Minora - E-commerce Shoe Website

Minora is a modern e-commerce frontend for a shoe store built using **React.js**, **TypeScript**, and **CSS**.  
It focuses on clean UI, component-based architecture, and a responsive shopping experience.

---

## 🚀 Features

- 🛍️ Product listing (Shoes collection)
- 🔍 Product details view
- 🛒 Add to Cart functionality
- ➕ Increase / Decrease quantity
- 💰 Cart total calculation
- 📱 Fully responsive design
- ⚡ Fast UI using React functional components
- 🎯 Type safety using TypeScript

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Language:** TypeScript
- **Styling:** External CSS
- **Package Manager:** npm
